# Restaurant Data Analysis

## Project Overview

This project involves analyzing restaurant menu data to gain insights into sales, profitability, and item popularity. The dataset contains information about menu items, their prices, ingredients, and the restaurants offering these items. The analysis includes data cleaning, summarizing key metrics, and visualizing results.

## Dataset

The dataset `restaurant.csv` contains the following columns:
- `RestaurantID`: Unique identifier for each restaurant.
- `MenuCategory`: Category of the menu item (e.g., Appetizer, Main Course, Dessert).
- `MenuItem`: Name of the menu item.
- `Price`: Price of the menu item.
- `Ingredients`: List of ingredients used in the menu item.

## Objectives

1. **Data Cleaning**: Handle missing values, remove duplicates, and prepare the data for analysis.
2. **Data Analysis and Visualization**: Perform various analyses and visualizations to understand sales, profitability, and item popularity.

## Data Cleaning Steps

1. **Handle Missing Values**: Checked for and handled any missing values.
2. **Remove Duplicates**: Removed duplicate entries to ensure data accuracy.
3. **Format Ingredients**: Cleaned and formatted the `Ingredients` column.
